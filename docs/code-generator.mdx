# Description

The **Code Generator** feature in Studio Pro is revolutionizing the way users tackle code-related problems. This innovative tool empowers users to streamline their coding process and overcome complex challenges by automatically generating optimized code snippets and solutions.

The Code Generator feature empowers users to ask code-related questions to OpenAI's advanced language model, **GPT**, and receive code feedback in a matter of seconds. Whether you're a seasoned developer or new to coding, this feature is designed to make development quicker, easier, and more accessible for everyone.

<img className="block" src="images/files_readme_io/_f7ebc78_image.png" alt="" />

With the Code Generator, users can now generate code snippets for various tasks within ElectroNeek Studio Pro. The feature is seamlessly integrated into the Assign to Variable, Execute Code, and Command Prompt functionalities, expanding its usability across different areas of the platform.

Let's explore how the Code Generator works in practice.

1.  Imagine you need to perform a task in the Command Prompt that involves sorting out a particular set of elements.
2.  Instead of manually writing the code, you can simply ask the AI to generate the necessary code snippet by making a code-related query.
3.  The Code Generator will then automatically generate the corresponding command prompt code for you. This capability empowers users with limited command prompt knowledge to achieve their automation goals effortlessly.

# How to use it

1.  To use Code Generator in Studio Pro, start by adding an **["Execute code"](/docs/others)
    ** activity to your canvas. You can find it under the _Activities tab →Execute JS Code_. Double-click it or drag-and-drop it to the canvas.

<img className="block" src="images/files_readme_io/_ed9c230_image.png" alt="" />

2.  **Optional:** You can add as many variables as you need and name them from the **"Properties"** tab. It is not necessary to initialize them before proceeding.

<img className="block" src="images/files_readme_io/_6a2f6bb_image.png" alt="" />

3.  Within **"Execute JS Code"** Properties tab, you will find the Code Generator.

<img className="block" src="images/files_readme_io/_dbe2afb_image.png" alt="" />

4.  A new window will pop up.

<img className="block" src="images/files_readme_io/_e410842_image.png" alt="" />

5.  **Optional:** You can choose either input variables or output variables based on the functionality you wish to develop.

<img className="block" src="images/files_readme_io/_85d5879_image.png" alt="" />

6.  Provide a clear description of the code you need and Send.

<img className="block" src="images/files_readme_io/_452c20f_image.png" alt="" />

7.  Receive answer and Save.

<img className="block" src="images/files_readme_io/_d17e6a0_image.png" alt="" />

8.  This functionality will be added to your workflow.

<img className="block" src="images/files_readme_io/_0403675_image.png" alt="" />
